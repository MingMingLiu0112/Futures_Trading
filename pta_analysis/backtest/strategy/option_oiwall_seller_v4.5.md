## 三、密度加权仓位

```python
if self.density_weighted and density > 1.5:
    density_w = 1.0 + (density - 1.5) * 0.3  # density=1.5→1.0x, density=3.0→1.45x
    density_w = max(0.7, min(1.5, density_w))   # 限制0.7~1.5倍
    vol = max(1, min(20, int(vol * density_w)))
```
