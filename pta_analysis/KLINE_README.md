# PTA期货1分钟K线图功能

## 功能概述

本功能实现了PTA期货的1分钟K线图显示，包含以下核心功能：

### 1. 主图功能
- **K线显示**：支持1分钟、5分钟、15分钟、30分钟、60分钟多种时间周期
- **移动平均线**：MA5、MA10、MA20均线显示
- **人工划线工具**：
  - 趋势线（支持任意角度绘制）
  - 水平线（支撑/阻力位标记）
  - 垂直线（时间标记）
  - 文本标注（自定义文本标记）
  - 标注管理（删除、导出、导入）

### 2. 技术指标
- **MACD指标**：
  - 可配置参数（快线、慢线、信号线周期）
  - MACD柱状图显示
  - **MACD面积值计算**：计算正负柱状图面积及比例
- **KDJ指标**：
  - 可配置参数（周期、K平滑、D平滑）
  - K、D、J三线显示
  - 0-100范围显示，适合超买超卖分析

### 3. 用户界面
- 实时价格信息显示
- 涨跌幅度计算
- 成交量显示
- 数据更新时间显示
- 响应式设计，适配不同屏幕尺寸

## 技术架构

### 前端技术栈
- **Chart.js**：图表渲染引擎
- **Bootstrap 5**：UI框架
- **Font Awesome**：图标库
- **Vanilla JavaScript**：核心逻辑

### 后端技术栈
- **Flask**：Web框架
- **SQLite**：数据存储（可选）

## 使用方法

### 1. 启动应用
```bash
cd pta_analysis
python3 web_app.py
```

### 2. 访问K线图
在浏览器中访问：`http://服务器IP:8423/kline`

### 3. 基本操作

#### 切换时间周期
- 点击顶部的时间周期按钮（1分钟、5分钟、15分钟、30分钟、60分钟）
- 数据会自动刷新并重新计算指标

#### 使用绘图工具
1. 选择绘图工具：
   - 📏 趋势线：点击起点和终点绘制趋势线
   - ➖ 水平线：点击价格位置绘制水平线
   - | 垂直线：点击时间位置绘制垂直线
   - 📝 文本标注：点击位置添加文本标注
   - 🗑️ 删除：点击标注进行删除
   - 🧹 清除所有：清除所有标注

2. 调整绘图样式：
   - 颜色选择器：选择线条/文本颜色
   - 线条宽度：调整线条粗细（1-5）
   - 线条样式：选择实线、虚线、点线
   - 文本内容：输入标注文本

#### 调整指标参数
1. **MACD参数**：
   - 快线周期（默认12）
   - 慢线周期（默认26）
   - 信号线周期（默认9）

2. **KDJ参数**：
   - 计算周期（默认9）
   - K值平滑（默认3）
   - D值平滑（默认3）

3. 点击"应用指标"生效，点击"重置默认"恢复默认参数

#### 数据管理
- **刷新数据**：手动刷新最新数据
- **导出标注**：将当前所有标注导出为JSON文件
- **导入标注**：从JSON文件导入标注

## API接口

### 获取K线数据
```
GET /api/kline_data
参数：
- period: 时间周期（1,5,15,30,60），单位分钟
- symbol: 品种代码（默认TA0）

响应：
{
  "success": true,
  "symbol": "TA0",
  "period": "1",
  "data": [...],
  "count": 100,
  "timestamp": "2026-04-08T16:43:00"
}
```

## 特色功能

### 1. MACD面积值计算
- 自动计算MACD柱状图的正负面积
- 显示面积比例，帮助判断多空力量对比
- 实时更新，随数据变化自动重新计算

### 2. 多时间周期支持
- 1分钟：高频交易分析
- 5分钟：短期趋势分析
- 15分钟：中期趋势分析
- 30分钟：中长期分析
- 60分钟：日內趋势分析

### 3. 标注持久化
- 标注可导出为JSON文件
- 支持从JSON文件导入标注
- 标注包含时间戳和样式信息

### 4. 实时数据更新
- 价格信息实时显示
- 涨跌幅自动计算
- 成交量动态更新

## 故障排除

### 常见问题

1. **图表不显示**
   - 检查Chart.js是否加载成功
   - 查看浏览器控制台是否有JavaScript错误
   - 确认API接口返回数据格式正确

2. **数据不更新**
   - 检查网络连接
   - 确认后端服务正常运行
   - 查看浏览器开发者工具中的网络请求

3. **绘图工具不工作**
   - 确认已选择正确的绘图工具
   - 检查是否有JavaScript错误
   - 尝试刷新页面重新初始化

### 调试方法
1. 打开浏览器开发者工具（F12）
2. 查看Console标签页的错误信息
3. 查看Network标签页的API请求响应
4. 检查Elements标签页的DOM结构
