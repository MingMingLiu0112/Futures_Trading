# 天勤网关合约JSON文件自动生成工具使用说明

## 概述

`generate_contracts.py` 是一个自动化工具，用于从天勤API查询所有可用合约，并自动生成符合天勤SDK标准格式的JSON配置文件。

## 功能特性

### 1. 自动查询合约
- 从天勤API查询所有可用合约
- 自动识别交易所和品种
- 支持期货和期权合约

### 2. 智能分类
- **期货合约**：自动识别并生成期货合约配置
- **期权合约**：根据天勤SDK标准格式生成期权合约
  - 使用 `-` 分隔符（如 `v2603-C-4400`）
  - 支持多个行权价
  - 支持多个到期月份

### 3. JSON格式输出
- 生成标准的JSON格式配置文件
- 支持中文编码（UTF-8）
- 格式化输出，便于阅读

## 使用方法

### 步骤1：准备环境

确保已安装天勤SDK库：
```bash
pip install tqsdk
```

### 步骤2：运行脚本

在命令行中运行：
```bash
python generate_contracts.py --account <天勤账号> --password <天勤密码>
```

**参数说明**：
- `--account`: 天勤账号（必填）
- `--password`: 天勤密码（必填）

### 步骤3：查看输出

脚本会输出以下信息：
1. 查询到的合约数量
2. 生成的期货合约数量
3. 生成的期权合约数量
4. 保存的文件路径
5. 总合约数量

### 步骤4：使用生成的文件

将生成的 `tqsdk_contracts.json` 文件复制到：
```
vnpy/gateway/tqsdk/tqsdk_contracts.json
```

然后重启天勤网关，网关会自动加载新的合约配置。

- 玉米期权（c2605-C-2200）

- 棉花期权（CF605-C-11000）

## 期权合约格式说明

根据天勤SDK官方文档，期权合约使用以下格式：

### 大商所（DCE）
```
DCE.v2603-C-4400  - PVC期权看涨
DCE.v2603-P-4400  - PVC期权看跌
```

**格式说明**：
- `DCE`: 交易所代码
- `v2603`: 品种代码 + 到期月份
- `-`: 分隔符
- `C`: 看涨期权
- `4400`: 行权价

### 上期所（SHFE）
```
SHFE.au2004C308  - 黄金期权看涨
SHFE.au2004P308  - 黄金期权看跌
```

### 中金所（CFFEX）
```
CFFEX.IO2002-C-3550  - 沪深300股指期权看涨
CFFEX.IO2002-P-3550  - 沪深300股指期权看跌
```

## 注意事项

1. **天勤账号密码**：请确保使用正确的天勤账号和密码
2. **网络连接**：确保网络连接正常
3. **API权限**：确保账号有查询合约的权限
4. **文件备份**：生成新文件前，建议备份原有的 `tqsdk_contracts.json`
5. **网关重启**：生成新文件后，需要重启天勤网关才能生效

## 故障排查

### 问题1：查询失败
**症状**：提示"未查询到合约，使用示例合约生成"
**原因**：天勤API连接失败或账号密码错误
**解决方法**：
- 检查天勤账号和密码是否正确
- 检查网络连接
- 检查账号是否有查询合约权限

### 问题2：生成的合约不完整
**症状**：某些合约缺失或格式不正确
**原因**：天勤API返回的合约列表不完整
**解决方法**：
- 手动检查天勤官方软件中的合约列表
- 根据实际需要手动补充缺失的合约

### 问题3：期权合约无法识别
**症状**：期权模块无法识别期权合约
**原因**：期权代码格式不符合天勤SDK标准
**解决方法**：
- 确认期权代码格式使用 `-` 分隔符
- 检查期权代码是否包含 `-` 字符
- 参考[天勤SDK官方文档](https://doc.shinnytech.com/tqsdk/latest/)
