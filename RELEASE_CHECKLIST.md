## 🔙 回滚方案

### 快速回滚命令
```bash
# 停止新服务
systemctl stop futures-trading-new

# 切换回稳定版本
git checkout v2.10.1-stable
git push origin main

# 重启服务
systemctl restart futures-trading

# 验证回滚成功
curl http://47.100.97.88/api/health
```

### 回滚触发条件
- API错误率持续5分钟超过5%
- 响应时间持续超过5秒
- 核心功能不可用
- 用户反馈严重问题
