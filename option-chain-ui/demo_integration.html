集成方法
方法一：iframe嵌入 (推荐)

将组件作为独立应用部署，通过iframe嵌入到PTA平台。

<!-- 在PTA平台页面中添加 -->
<div class="option-chain-container">
  <iframe
    src="/static/option-chain/index.html"
    width="100%"
    height="800px"
    frameborder="0"
    scrolling="no"
  ></iframe>
</div>
简单 独立部署 低风险
方法二：组件直接集成

将React组件直接集成到PTA平台的前端代码中。

方法三：微前端架构

使用微前端框架（如qiankun）集成，实现技术栈完全独立。

// 主应用注册子应用
registerMicroApps([
  {
    name: 'option-chain',
    entry: '//localhost:3000',
    container: '#option-chain-container',
    activeRule: '/options/chain'
  }
]);
复杂 技术独立 可扩展
组件特性
  • T型布局：左侧看涨，右侧看跌
  • 简化字段：价格、持仓量变化%、成交量变化%、隐波变化、希腊字母
  • 实时数据更新（支持WebSocket）
  • 交互式控制面板
  • 平值期权自动高亮
  • 多种排序方式（行权价、成交量、持仓量、隐波）
  • 字段显示/隐藏控制
  • 响应式设计（支持移动端）
  • 市场统计面板
  • 与PTA平台数据无缝对接
快速开始
安装与运行
  1. 下载组件代码到PTA平台目录
  2. 运行安装脚本：./start.sh
  3. 选择选项1启动开发服务器
  4. 访问 http://localhost:3000 查看组件
  5. 按照集成指南进行部署
数据对接

组件支持多种数据源：

  • 模拟数据：用于开发和测试
  • PTA平台API：连接现有数据源
  • 天勤量化API：实时市场数据
  • 自定义数据源：通过配置文件对接
